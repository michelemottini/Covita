Visualizzazione dati epidemia COVID-19

Visualizza i dati pubblicati giornalmente dalla Protezione civile: nazionali, per regione o provinciali.

Per le singole province è disponibile solo il numero totale di casi. Per tutti i dati è possibile visualizzare i valori totali o gli aumenti (diminuzioni) giornalieri, calcolati sottraendo dal totale del giorno il totale del giorno prima. Ogni dato può essere nascosto / visualizzato clickando sull'etichetta qui sotto.
